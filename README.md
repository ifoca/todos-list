# To-Dos List project

This is an individual learning project, created as part of exploring the React useState() hook

## Description

- single page application implementing a simple to dos list
- uses local storage for saving entries
- renders the todo items with useState

## Installation steps

    - clone the repository
    - run `npm install` to install all the dependencies
    - run `npm run dev` to create the local environment
    - access the local environment at `http://localhost:5173/`
